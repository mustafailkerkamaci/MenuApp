# Asian Kitchen's Menu

Bu proje, JavaScript kullanılarak dinamik olarak oluşturulmuş bir Asya restoran menüsü uygulamasıdır. Menü verileri bir dizi içinde tutulur ve kullanıcı seçtiği kategoriye göre içerik filtreleyebilir.

## Özellikler

- Menü verileri JavaScript dizisi üzerinden dinamik olarak oluşturulur
- Kategoriye göre filtreleme (All, Korea, Japan, China)
- `map`, `reduce` ve `filter` metotları kullanılmıştır
- Responsive tasarım
- Stock görseller internet üzerinden dinamik olarak alınmaktadır
- HTML, CSS ve JavaScript ile geliştirilmiştir

## Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Bootstrap (grid yapısı için)
- Google Fonts

## Kurulum

1. Proje dosyalarını bilgisayarına indir.
2. `index.html` dosyasını tarayıcıda aç.
3. Menü otomatik olarak yüklenecektir.

## Dosya Yapısı

- `index.html` - Ana sayfa
- `style.css` - Stil dosyası
- `app.js` - JavaScript menü ve filtreleme işlemleri
